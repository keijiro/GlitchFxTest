# KinoGlitch

**KinoGlitch** is a Unity URP renderer feature package that provides analog and
digital glitch post-processing effects.

# System requirements

- Unity 6000.0 or later
- Universal Render Pipeline (URP)

# How to use

1. Add **AnalogGlitchRendererFeature** and/or **DigitalGlitchRendererFeature** to your URP Renderer Asset.
2. Add **AnalogGlitchController** and/or **DigitalGlitchController** to a Camera.
